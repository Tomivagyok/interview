## Project Structure

- **`runners/TestRunner.java`**: Main class to execute all Cucumber scenarios.
- **`stepDefinitions/`**: Contains all step implementation classes.
- **`dto/`**: Contains POJO classes for DataTable mapping.
- **`features/`**: Contains `.feature` files defining test scenarios.

---

## Running the Tests

The tests are executed using the **TestRunner** class.

1. Navigate to the project root.
2. Run the TestRunner class in your IDE, or use Maven from the command line: mvn test
